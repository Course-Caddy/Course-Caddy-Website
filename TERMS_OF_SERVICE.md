## Description of Service

Course Caddy is a mobile application that helps golfers calculate adjusted club distances based on environmental conditions including temperature, elevation, and humidity.

- Environmental condition adjustments (manual entry)
- Baseline variable configuration
- Distance calculations based on temperature, elevation, and humidity

### Pro Features (Requires $2.99/month Subscription):

- Wind calculator
- Print custom yardage cards
- Multiple golf bag storage (up to 5)
- Real-time weather updates every 30 minutes
